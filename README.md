# per compilare:

```sh
aarch64-linux-gnu-gcc -o NomeFileOutput -static NomeFileSorgente
```

# per avviare 
(ricorda che, a meno che tu non sia su arm64, abbiamo bisogno di un emulatore):

```sh
qemu-aarch64 NomeFileOutput
```
